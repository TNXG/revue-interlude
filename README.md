# Revue Interlude

> “它终将上演……那一日的星光。”

一个受《少女歌剧 Revue Starlight》启发的个人主页项目。

## 特性

- **电影级体验**：由 GSAP 驱动的流畅滚动叙事体验。
- **双语支持**：无缝呈现日语和中文内容。
- **响应式设计**：兼顾桌面的宏大视觉与移动端的亲密体验。
- **隐藏彩蛋**：为有心人准备的隐藏层。

## 技术栈

- **框架**: [Nuxt 3](https://nuxt.com)
- **UI 库**: [Vue 3](https://vuejs.org)
- **动画**: [GSAP](https://greensock.com/gsap)
- **样式**: SCSS
- **包管理器**: pnpm

## 安装与设置

请确保安装依赖：

```bash
# pnpm
pnpm install
```

## 开发服务器

在 `http://localhost:3000` 启动开发服务器：

```bash
pnpm dev
```

## 生产环境

构建生产环境应用：

```bash
pnpm build
```

本地预览生产环境构建：

```bash
pnpm preview
```
